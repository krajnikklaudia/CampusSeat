# CampusSeat - University Facility Booking System

## Overview
This repository contains the source code for the frontend UI prototype of the CampusSeat application, created for the Software Engineering course.

## Technologies Used
- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript

## How to Run the Prototype
Since this is a frontend-only prototype, no local server or backend installation is required. To run the application, follow these simple steps:

1. Extract the downloaded `CampusSeat_SourceCode.zip` archive to a local folder.
2. Navigate to the extracted folder.
3. Double-click the `index.html` file. 
4. The prototype will automatically open and run in your default web browser (Google Chrome, Mozilla Firefox, Safari, or Microsoft Edge are recommended).

**Note:** An active internet connection is required when opening the file, as the project loads the Tailwind CSS framework dynamically via a CDN link.
